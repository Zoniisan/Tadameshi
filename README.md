# タダ飯データベース
2018年度京大新歓タダ飯データベース

## localで起動したい
1. settings.pyのif 'Zoneesan' in HOSTNAMEのZoneesanを自分のホスト名に変える
2. python manage.py collectstatic → migrate → runserver

## remoteで起動したい
env.mdに従って環境を構成してください。
ただしこの資料は別のプロジェクトをすすめる際に書いたものなので
アプリ名がpenguinだったりユーザー名がzuyaだったりします。
適宜読み替えてください。
